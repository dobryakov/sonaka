Суть проекта - личный кабинет покупателя интернет-магазина Sonaka. Магазин продаёт товары для сна.

Технологический стек:
- docker
- backend: ruby on rails (предоставляет только REST API)
- frontend: react, как одностраничное приложение (SPA), пользуется REST API с бэкэнда
- database: postgresql

Фирменный стиль:
- мягкие светло-зелёные цвета
- трендовые анимации
- glass-morphism

Прочее:
- веб-интерфейс на русском языке
- активно используй автоматизированное тестирование, в том числе пользовательского интерфейса
- для запуска проекта составь отдельный набор тестовых данных и загрузи их в БД

Бизнес-логика проекта:
- личный кабинет покупателя интернет-магазина
- покупатель видит дашборд 
    - со списком своих заказов и общей суммой заказанного
    - со списком всех ранее купленных товаров подряд (каждый товар с указанием количества)
- покупатель может зайти в каждый заказ и увидеть детали (перечень товаров с картинкой, названием товара, количеством и ценой)
